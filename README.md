# Interview SWAPI
Project was copied from https://github.com/phalt/swapi

# Befor first run
```bash
python3 -m venv venv
. venv/bin/activate
make install
make build
make load_data
```

# How to run?
```bash
make serve
```
SWAPI should be available under below url:
```bash
http://localhost:5000
```
